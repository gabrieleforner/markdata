# MarkData

## Description

MarkData è un software per mantenere traccia e eseguire analisi di base su voti come la <br>
media, voto più frequente e grafici per mantenere graficamente sotto controllo la situazione <br>
Il programma è disponibile su Desktop sui seguenti sistemi operativi:
*  macOS
*  Windows
*  GNU/Linux Distros

# Building instructions
1. Obtain a copy of Flutter SDK from https://docs.flutter.dev/get-started/install
2. Clone locally this project and ensure Flutter SDK is working trough `flutter doctor`
3. Move in the project directry where the project has been cloned
4. Run `flutter build <destination platform>`
