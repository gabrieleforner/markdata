# MarkData

## Description

MarkData is a program for desktop platforms to perform base data analysis (in the following photo is applied <br>
for school marks tracking) like mean value, frequency and median values. This project it's <br>
written in Dart with the Flutter framework to ensure support on:
*  macOS
*  Windows
*  GNU/Linux Distros

# Building instructions
1. Obtain a copy of Flutter SDK from https://docs.flutter.dev/get-started/install
2. Clone locally this project and ensure Flutter SDK is working trough `flutter doctor`
3. Move in the project directry where the project has been cloned
4. Run `flutter build <destination platform>`
